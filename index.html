Sensors Library

Servo Motor

A motor that moves to a specific position using control signals. It uses a feedback mechanism to achieve accurate positioning and is commonly employed in robotics, CNC machinery, camera systems, and automation applications.

Ultrasonic Sensor

Measures distance by sending ultrasonic waves and detecting echoes. Commonly used in robotics, automation, and obstacle detection systems.

Distance_IR

An infrared distance sensor uses infrared light to measure the proximity of objects. It emits an infrared beam and detects the reflected light to calculate distance.

QtPy

The QtPy is a compact and versatile microcontroller board designed for quick prototyping and small-scale projects. Ideal for wearable devices, IoT applications, and compact embedded systems.

ItsyBitsy ATmega32u4

The ItsyBitsy ATmega32u4 is a miniature microcontroller board. Perfect for space-constrained projects like wearable devices, custom keyboards, or portable electronics.

Linear Potentiometer (slider)

A linear potentiometer is a variable resistor that adjusts resistance linearly based on the slider's position. It is commonly used to control voltage or signal levels, such as adjusting volume or brightness.

Linear Potentiometer (soft)

A linear potentiometer is a variable resistor that adjusts resistance linearly based on the slider's position. It is commonly used to control voltage or signal levels, such as adjusting volume or brightness.

Long Flex Sensor

A long flex sensor is a variable resistor that changes resistance based on bending or flexing. It is commonly used to detect movement, angles, or bending gestures in applications like wearable devices, robotics, and motion-controlled interfaces.

Raspberry Pi

A Raspberry Pi is a small, affordable single-board computer designed for educational and hobbyist purposes. It is equipped with USB ports, HDMI output, and GPIO pins, making it ideal for programming, electronics projects, IoT applications, and as a media center.

MicroSD Memory Card

A MicroSD Memory Card is a compact storage device used to store data, including operating systems, applications, and files. It is essential for devices like Raspberry Pi as it serves as the primary storage for the operating system and user data.
